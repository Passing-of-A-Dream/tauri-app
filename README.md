# tauri-app

## 先决条件
  - 安装window桌面开发环境，C++库
  - 安装rust
  - 安装node.js
  - 安装pnpm
  - 有问题请参照终端输出给出的链接
  
## 启动方法
  - pnpm i
  - pnpm add -D @tauri-apps/cli
  - pnpm tauri init
  - pnpm tauri dev
